# Tapioca discourse

## Installation
```
pip install tapioca-discourse
```

## Documentation
``` python
from tapioca_discourse import Discourse


api = Discourse(api_root='https://try.discourse.org', api_key='API_KEY', api_username='API_USERNAME')

```
